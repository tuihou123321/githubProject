# 基于vue+MongoDB+爬虫的全栈后台项目-Girl

## 项目说明 
爬虫后台管理系统，可视化查看爬虫配置状态，代理IP配置，响应式设计支持手机端
数据来源于爬虫


**项目地址**：https://github.com/Xu-Angel/Girl
**在线演示**：http://girl.xutianshi.top/admin/index.html#/dashboard

### 技术栈
- vue
- express
- MongoDB
- mongoose
- element-ui
