# 基于nextjs打造的移动端同构贷款项目-react-ssr


## 项目说明 
使用nextjs构建的一个ssr同构项目，使用了服务器html缓存，和静态资源CDN，大大提高页面速度；
首屏渲染在0.6s左右；

**项目地址**：[https://github.com/minooo/react-ssr](https://github.com/minooo/react-ssr)

**线上地址**：[http://m.jr.duduapp.net/](http://m.jr.duduapp.net/)


### 技术栈
- react
- next.js4   //此版本有点老了
- koa
- antd-mobile
- redux
- lru-cache


### 启动
```javascript
npm i 
npm run dev
```
访问： http://localhost:8868
